##  DSA Practice

This repository is a personal sandbox for **Data Structures and Algorithms** implementations in Java. It’s built to be a hands-on, evolving collection of problem-solving techniques and algorithmic patterns.

---

### 📂 Repository Overview

The project is organized by topic within the `src/main/java` directory. Each package represents a different category of DSA, starting with foundational sequences and moving toward complex structures.

* **Current Focus:** Mathematical Algorithms (Fibonacci, etc.)
* **Upcoming:** Sorting, Searching, Linked Lists, and Trees.

---

### 💻 Get the Code

If you want to explore the implementations, see the logic, or run the algorithms yourself, simply clone the repository and check out the source files:

1. **Clone the repo:**
```bash
git clone https://github.com/Delaksan-Sritharan/dsa-practice.git

```


2. **Explore the source:**
   Navigate to `src/main/java/` to find the specific algorithm you're looking for.

---

### 🛠️ Environment

* **Language:** Java
* **Build Tool:** Maven

> **Note:** This is a "code-first" repository. The best way to understand the logic is to pull the latest changes and dive into the classes!
